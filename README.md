# 图片转富文本工具![Python](https://img.shields.io/badge/Python-3.7%2B-blue) ![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey) ![License](https://img.shields.io/badge/License-MIT-green) 

本工具可将任意图片转换为支持颜色和字体大小的富文本字符串（基于 `<color>` 和 `<size>` 标签），适用于支持此类标签的聊天软件、游戏内文本系统等场景。输出结果以分段形式展示，便于复制粘贴使用。

 ## ✨ 功能特点 

- 支持 PNG、JPG、BMP、GIF 等常见图片格式 - 可自定义目标分辨率（支持保持宽高比） 

- 透明像素处理选项：  

  - 保留原始透明效果  

  -  替换为自定义背景色 


- 富文本智能分段功能：
  - 按像素数量分段（默认每段 40 像素）  
  - 按字符长度分段（默认每段 1000 字符），避免超出平台限制
- 可选设置文本字体大小
-  实时生成预览图，直观查看转换效果 -
- 分页展示结果，每段标注所属行号与段序 -
- 一键复制任意段落到剪贴板 
 ## 🛠️ 依赖库 

Python 3.7+ 

PyQt5

Pillow (PIL)

pyperclip 

安装依赖： 

```
pip install PyQt5 Pillow pyperclip
```

## ▶️ 使用方法

```
python main.py
```

1. 在界面中：
   - 选择一张图片
   - 设置目标分辨率（应避免输出过长）
   - 选择透明像素处理方式
   - 选择分段规则（推荐“按字符分段”以兼容大多数平台）
   - （可选）设置字体大小
2. 点击 **“开始生成”**
3. 处理完成后可：
   - 点击 **“预览结果”** 查看渲染效果
   - 浏览分页结果，点击 **“复制”** 按钮复制任意段落

## 📝 注意事项

- 本工具生成的富文本依赖目标平台支持 `<color>` 和 `<size>` 标签（如部分 Unity 游戏、Discord 插件等）。
- 若用于不支持颜色标签的环境，请勿使用此工具。
- 字符分段模式会智能合并连续同色像素，减少标签数量，同时严格控制总字符数。
